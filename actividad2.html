Conducción Controlada por Aplausos

Objetivos de la Actividad

En esta actividad, los estudiantes programan Edison usando un código de barras que activa un programa que les permite conducir a Edison aplaudiendo.

Cómo Funciona

El programa de conducción controlado por aplausos utiliza el sensor de sonido del robot Edison para detectar sonidos fuertes, como aplausos.

El programa preestablecido hará que Edison gire hacia la derecha cuando se detecte un aplauso y hará que Edison avance durante unos segundos cuando se detecten dos aplausos.

Consejos y Trucos

• Puede ser útil revisar los componentes de Edison, enfatizando la ubicación del sensor de sonido de Edison antes de que los estudiantes comiencen la actividad.

 Puede consultar las imágenes en la actividad uno. Para obtener una guía técnicamente más precisa, descargue el Guía de diseño de la placa base Edison.

• Los robots pueden tener dificultades para detectar sonidos cuando hay un alto nivel de ruido de fondo.

 Hacer que los estudiantes toquen con un dedo cerca del sensor de sonido de su Edison creará el mismo efecto que aplaudir.

Extensiones de la Actividad

1. Construye un circuito de carreras físico con barreras y giros para navegar.

 Esto puede crear un desafío adicional para los estudiantes, requiriendo que utilicen las entradas variables de aplausos simples y dobles para controlar a Edison.

2. Haga una pista de carreras con un giro a la izquierda, animando a los estudiantes a usar habilidades creativas para resolver problemas.

3. Experimente para aprender qué otros sonidos además de aplaudir y golpear Edison puede detectar.

 Analice si alguno de estos sonidos es práctico para controlar a Edison.
